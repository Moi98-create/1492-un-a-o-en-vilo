Situación de Aprendizaje / Proyecto: **1492: Un año en vilo**

Este repositorio contiene el **Recurso Educativo Abierto (REA)** desarrollado íntegramente con la herramienta de software libre **eXeLearning 4.0** para el curso de formación docente.

##  Descripción del Proyecto
Este recurso ha sido diseñado como una herramienta base para el trabajo directo con el alumnado de **2º ESO**. A través de un enfoque motivador, competencial y adaptado a las pautas del **Diseño Universal para el Aprendizaje (DUA)**, los estudiantes abordarán tareas clave para la gestión de su identidad digital, comunicación y colaboración en la sociedad del conocimiento.

## Contenidos Incluidos (Estructura eXeLearning)
El recurso está organizado para ofrecer una navegación clara y accesible, incluyendo:
*   **Presentación y descripción** detallada del proyecto y sus objetivos.
*   **Información legal y cívica** sobre la protección de datos personales y garantías de derechos digitales en el entorno escolar.
*   **Guía Didáctica** para el profesorado con la vinculación curricular (tareas frente a criterios de evaluación LOMLOE).
*   **Paneles de curación de contenidos** integrados (tanto para soporte docente como para la investigación del alumnado).
*   **Línea temporal** interactiva para la organización cronológica de las tareas.
*   **Evaluación formativa:** Criterios claros e instrumentos de evaluación integrados (rúbricas/dianas de autoevaluación).

## Descarga y Reutilización (Formato Editable)
En la última sección del recurso (*Nodo de Créditos*) se encuentra disponible el botón de descarga del archivo fuente original con extensión **`.elp`**. 
Cualquier docente puede descargar este archivo, abrirlo en su propia instalación de eXeLearning y adaptarlo o modificarlo libremente según las necesidades de su aula.
